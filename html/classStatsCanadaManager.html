Detailed Description

An object that will retrieve data from Statisitcs Canada.

It calls the RequestManager object and makes a POST request to get data. This object will only work for Statistics Canada as it is designed to parse their returned data and make a POST body designed for Statistics Canada. To make a more general POST request, look at the 'RequestManager' object.

Anthony Lin
