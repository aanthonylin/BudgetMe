Detailed Description

An object that will retrieve data specifically for BudgetMe.

It is an object with a specialized purpose. It retrieves data and meta data from Statistics Canada that will be required for BudgetMe. The data returned will only work with BudgetMe or any application that requires the same data. If a more general object is needed, use the StatsCanadaManagerObject.
