\doxysubsection{Detailed Description}
An object that will retrieve data from Statisitcs Canada. 

It calls the \mbox{\hyperlink{classRequestManager}{Request\+Manager}} object and makes a P\+O\+ST request to get data. This object will only work for Statistics Canada as it is designed to parse their returned data and make a P\+O\+ST body designed for Statistics Canada. To make a more general P\+O\+ST request, look at the \textquotesingle{}\mbox{\hyperlink{classRequestManager}{Request\+Manager}}\textquotesingle{} object.
